Projects

Mask detection using images

This was a project I wanted to do during covid-19 lockdown period. I used python with libraries like tensorflow, keras and OpenCV . The basic aim of the project is to detect the presence of a face mask on human faces on images. I have used deep learning to develop the face detector model.The Github repository for the project.

Chicago crime data analysis

This project was a part of the Remarkskill internship and was done under our mentor's guidance. The analysis was for Chicago crime data from 2015 to 2020.
The analysis included results depicting relations in crime, time, location etc. features , other than this predictions were made for the coming months.The Github repository for the project.

Iris data analysis

This project was a part of the internship at The Sparks Foundation. The basic aim of the project was to visualize the iris dataset in various ways and then apply various classification models. The accuracies are - for decision tree 96.67%, for gaussian naive bayes 96.67% and for SVC it is 100%. The Github repository for the project.
